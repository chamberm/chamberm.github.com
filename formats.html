File formats

 
    Nifti Anatomies (*.nii | *.nii.gz)
  • Has to be either FLOAT32, INT16 or short int (e.g., scalar maps such as T1, T2, FA, fMRI activations, etc.
    [X Y Z 1] or [X Y Z 3] for RGB maps where 3 represents the main tensor direction stored in a 256 color basis).
  • Note that the FiberNavigator doesn't takes into account the affine part of the matrix.
    Glyphs (*.nii | *.nii.gz)
  • Tensors [X Y Z 6] where the order of symmetric tensor terms is xx, xy, xz, yy, yz, zz.
  • ODFs (MRtrix basis, or DIPY basis). For example, a [X Y Z 45] file where 45 is the SH coefficients of order 8.
  • Peaks (extracted from fODFs) [X Y Z 9] where 9 here indicates 3 main peaks, each having xyz coordinates.
    Streamlines
  • *.tck from MRtrix (world space coordinates, RAS+ mm)
  • *.trk from TrackVis (convention varies, simple double check that they fit with the anatomy)
  • *.fib (Fibernavigator format, a simple binary VTK file with points in image space, voxel mm)
  • *.vtk (same as above, but world coords)
